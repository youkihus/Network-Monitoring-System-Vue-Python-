网络状态监控项目 - 启动引导

欢迎使用网络状态监控项目！这是一个使用Vue.js前端和Python后端开发的网络监控工具，可以实时监控您的电脑网络状态、上传下载速度等信息。

请按照以下步骤启动项目：

1. 启动后端服务

  • 打开命令提示符（CMD）或PowerShell
  • 导航到项目的backend目录： 
    cd D:\phpStudy\phpstudy_pro\WWW\新建文件夹\自用\vue+python项目\网络监控系统\backend
  • 安装依赖包： 
    pip install -r requirements.txt
  • 启动后端服务： 
    python app.py

    • 2. 启动前端服务

  • 打开另一个命令提示符（CMD）或PowerShell窗口
  • 导航到项目的frontend目录： 
    cd D:\phpStudy\phpstudy_pro\WWW\新建文件夹\自用\vue+python项目\网络监控系统\frontend
  • 安装依赖包（首次运行需要）： 
    npm install
  • 启动前端开发服务器： 
    npm run serve

    • 3. 访问应用

    启动完成后，您可以通过浏览器访问以下地址使用网络状态监控应用：

    http://localhost:8080

    注意事项：

    • 确保您已经安装了Python和Node.js环境
    • 运行后端服务可能需要管理员权限以获取完整的网络信息
    • 如果遇到端口冲突，可以修改配置文件中的端口号

    4. 项目功能

    • 实时监控网络连接状态
    • 显示上传下载速度和总流量
    • 展示网络接口详细信息
    • 历史数据图表展示
    • 支持自动刷新和手动刷新

    如有问题，请参考项目目录下的README.md文件或联系开发者。
